# ZDailyScan — Design System

Dark UI com tokens OKLCH para consistência de cor em displays modernos.
Paleta principal: fundo escuro `oklch(0.13 0.02 260)`, superfícies em azul-acinzentado escuro, acento azul `oklch(0.70 0.18 250)`.
Tipografia Inter via Google Fonts; sem frameworks CSS externos.
Scores de produto exibidos como barras de progresso tricolor (verde/amarelo/vermelho).
